Descriptión

  This Class PHP allow create dynamic queries to create, update, delete and getting result for a database.

   Type of query you can run.

   INSERT INTO

   UPDATE

   DELETE

   SELECT FROM

   SELECT FROM JOIN

   SELECT FROM WHERE

   SELECT FROM WHERE LIKE

   SELECT FROM WHERE BETWEEN

   SELECT FROM WHERE ORDER BY

   SELECT FROM WHERE ORDER BY LIMIT

   QUERY( QUERY EXAMPLE : "SELECT * FROM"  );
